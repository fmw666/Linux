### 💬 切换目录命令——cd

+ **格式：**

    ```bash
    $ cd dir_path
    ```

+ **切换到上个目录下：**

    *不加参数：*

    ```bash
    $ cd
    ```

    *或者加上参数 `..`：*

    ```bash
    $ cd ..
    ```

    > 其中一个 `.` 表示当前路径

+ **绝对文件路径：**

    ```bash
    $ cd /usr/bin/
    $ pwd
    /usr/bin
    ```

    > 其中 pwd 命令用于查看终端下当前路径

+ **相对文件路径：**

    ```bash
    /usr$ cd bin/
    /usr/bin$ pwd
    /usr/bin
    ```
