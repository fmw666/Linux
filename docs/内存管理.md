### 💬 C 中变量、函数存储类型

&emsp;&emsp;三种模式：命令模式、插入模式、未行模式。

+ 命令模式：直接与 Linux 内核相交互

+ 插入模式：在命令模式下按下 `i` 进入，按下 `ESC` 退出

+ 未行模式：在命令模式下按下 `:` 进入，按下 `Enter` 退出
