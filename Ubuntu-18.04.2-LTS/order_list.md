#### 打开网易云音乐
```shell
sudo netease-cloud-music&
```

#### 关闭网易云音乐
```shell
sudo killall -9 netease-cloud-music
```

#### 删除文件
```shell
sudo rm -rf 文件名
```

#### 复制文件
```shell
sudo cp -r 该路径下的文件名 目的路径
```

#### 打开SQLite图形化工具
```shell
sqlitebrowser
```

#### 打开系统监控器
```shell
gnome-system-monitor
```

#### 打开MongoDB
```shell
sudo service mongod start
```

#### 关闭MongoDB
```shell
sudo service mongod stop
```
